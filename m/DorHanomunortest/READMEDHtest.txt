DORHANOMNORTEST. Doornik-Hansen Omnibus Multivariate (Univariate) Normality Test.
This file describe the multivariate version of the univariate omnibus test
for normality of Shenton and Bowman (1977), based on the transformed skewness
and kurtosis.

Usage

1. It is necessary you have defined on Matlab the X - data matrix. Size of
matrix (X) must be n-by-p; n=observations, p-independent variables=columns. 

2. For running this file it is necessary to call the rafisher function as 
DorHanomunortest(X,alpha). Please see the help DorHanomunortest.

3. Once you input the arguments, it will appears your results.
